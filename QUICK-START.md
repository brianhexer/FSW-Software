# Quick Start Guide

## 30-Second Setup
1. Open `script-editor.html` in your browser
2. Start typing your script
3. Your work auto-saves automatically

## Your First Script (2 Minutes)

```
INT. COFFEE SHOP - MORNING

Sarah sits alone at a corner table. She nervously sips her coffee.

The bell above the door chimes as James enters.

SARAH
I wasn't sure you'd come.

JAMES
Neither was I.

He sits down across from her.

SARAH
(pause, looking away)
So much has changed.

JAMES
Everything has changed.

FADE OUT.
```

**Try it:**
1. Paste this into the editor
2. Notice how formatting automatically detects
3. Press Ctrl+P to export as PDF
4. Click "Save As" and give it a title

## Example Patterns

### Pattern 1: Simple Dialogue Exchange
```
JOHN
Hello, how are you?

SARAH
I'm doing great, thanks for asking.

JOHN
That's wonderful to hear.
```

### Pattern 2: Action with Dialogue
```
INT. OFFICE - DAY

John walks to the window and looks out.

JOHN
The city looks different from here.

He turns back to Sarah.

JOHN (CONT'D)
Everything looks different from here.
```

### Pattern 3: Complex Scene
```
INT. RESTAURANT - NIGHT

Candles flicker on white tablecloths. Soft jazz plays.

Marcus and Elena sit across from each other, dressed formally.

ELENA
This is nice.

MARCUS
You deserve nice.

ELENA
(hesitant)
I need to tell you something.

Marcus leans forward, concerned.

MARCUS
What is it?

ELENA
I'm leaving the city tomorrow.

Long pause. Marcus's face falls.

FADE TO BLACK.
```

## Format Guide

### Scene Heading (Slugline)
- Starts with: INT., EXT., or INT/EXT.
- Auto-formatted to uppercase
- Example: `EXT. BROOKLYN BRIDGE - DUSK`

### Action
- Regular text, any case
- Describes what happens on screen
- Can be multiple lines

### Character
- Type in any case, displays as UPPERCASE
- Should be unique character names
- Example: `JOHN`, `SARAH`, `OFFICER #1`

### Dialogue
- Spoken words
- Automatically indented after character name
- Press Enter after character to start

### Parenthetical
- Action within dialogue
- Surrounded by parentheses
- Example: `(laughing)`, `(whispers)`, `(V.O.)`

### Transitions
- Ends with "TO:"
- Example: `CUT TO:`, `FADE IN:`, `DISSOLVE TO:`

## Key Tips

✅ **DO:**
- Use consistent character naming (same name = same character)
- Use uppercase for scene headings and character names
- Keep action descriptions clear and visual
- Break long scenes into logical beats

❌ **DON'T:**
- Mix character name variations (use "JOHN" not "John" and "JOHN")
- Write camera directions (use action instead)
- Use page breaks manually (they adjust automatically)
- Edit the exported PDF (export again if changes needed)

## Saving & Backup

- ✅ Auto-saves every 2 seconds (shows "Saved" status)
- ✅ Saves to browser localStorage (local computer only)
- ✅ Persists across page refreshes
- ✅ Manual save with custom title via "Save As" button

**To back up your script:**
1. Click "Export PDF" to save a formatted version
2. Or copy-paste script text to a text file
3. Or export to another format

## Exporting to PDF

1. **Quick Export:** Press Ctrl+P
2. **From Header:** Click "Export PDF" button
3. **Filename:** Based on your script title (spaces converted to hyphens)
4. **Format:** Perfect screenplay format, 12pt Courier Prime, ready to print

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Script not saving | Check browser allows localStorage (not private mode) |
| Formatting wrong | Ensure character names are ALL CAPS |
| Scene not detected | Scene headings must start with INT., EXT., or I/E. |
| PDF looks wrong | Try exporting again, check latest browser version |
| Lost my script | Check browser cache, try "Save As" to trigger save |
